LOGO INSTRUCTIONS
==================

I've embedded your logo directly into the HTML file, so it should work automatically!

HOWEVER, if you want to use your actual logo file:

1. Make sure "logo.jpeg" is in the SAME folder as your HTML files

2. The logo should automatically load

3. If it doesn't show, replace this line in EACH HTML file:

   <img src="data:image/jpeg;base64,..." 

   with:

   <img src="logo.jpeg"
